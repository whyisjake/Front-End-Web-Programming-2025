Learning Objectives

By the end of this course, students will be able to:

  • Construct semantic, standards-compliant web pages using HTML
  • Style web pages using CSS, including Flexbox and responsive design with media queries
  • Apply interactive behaviors using jQuery and basic JavaScript functions
  • Design and build functional, multi-page websites integrating structure, style, and interactivity
  • Use Git and GitHub for version control and collaborative development
  • Evaluate web interfaces based on usability, accessibility, and responsive design
  • Deploy websites to hosting environments with proper file structure
  • Collaborate effectively on team-based web development projects
